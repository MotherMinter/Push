# Push.
Send Minter coins with ease.

## Description

Minter Push wallet - simple light wallet
Backend part

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# migration
# yarn typeorm migration:generate -n InitMigration - generate new named (-n) migrations
yarn typeorm migration:run - run migration

# build app 
$ yarn build

# development
$ yarn start
```
